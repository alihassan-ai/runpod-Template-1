Place your ComfyUI workflow JSON files in this directory.

They will be automatically copied to:
/workspace/ComfyUI/user/default/workflows/

You can add workflows by:
1. Saving from ComfyUI interface
2. Downloading from community sources
3. Creating custom workflows

Popular workflow sources:
- ComfyUI Examples: https://comfyanonymous.github.io/ComfyUI_examples/
- OpenArt: https://openart.ai/workflows
- CivitAI: https://civitai.com/workflows

Note: Video generation workflows may require additional models
to be downloaded via ComfyUI Manager on first use.
